# Exercícios de Lógica de Programação em Python

Este repositório contém exercícios resolvidos da disciplina de Lógica de Programação, realizados por **Marcio Pedroso**.  
Os exercícios cobrem conceitos de entrada e saída de dados, estruturas de decisão, laços de repetição, funções, tratamento de exceções e manipulação de listas e dicionários em Python.

## Exercícios

### 1. App para calcular valor de parcelas e juros.
- Recebe o valor do pedido e número de parcelas feito pelo usuário.
- Calcula o juros acima de certa quantidade de parcelas e imprime o valor das parcelas e o total com juros.

### 2. App de Vendas de Marmitas
- Simula o pedido de marmitas com diferentes sabores e tamanhos.
- Utiliza menu interativo, validação de entradas, laços e acumulador de pedidos.

### 3. Sistema de Cobrança de Camisetas em Atacado
- Calcula o valor da venda de camisetas considerando modelo, quantidade, descontos e fretes.
- Utiliza funções, validação de entradas e tratamento de exceções.

### 4. Gerenciamento de Funcionários
- Permite cadastrar, consultar e remover funcionários usando menu interativo.
- Utiliza lista de dicionários, funções, busca e remoção por critérios.

## Como Executar

1. Certifique-se de ter o [Python 3](https://www.python.org/) instalado.
2. Clone este repositório ou baixe os arquivos.
3. No terminal, execute o exercício desejado:
   ```bash
   python nome_do_arquivo.py
   ```
   (Substitua pelo nome do exercício, ex: `exercicio1.py`)
